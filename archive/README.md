# 归档文件说明

本目录包含已归档的测试脚本、示例文件和文档。

## 目录结构

```
archive/
├── test-scripts/        # 测试脚本归档
│   ├── global/         # 全球区域相关测试
│   ├── proxy/          # 代理服务器测试
│   └── product/        # 商品相关测试
├── json-samples/       # JSON 示例文件归档
└── progress-docs/      # 进度文档归档
```

## 归档原因

### 测试脚本
- 临时测试脚本，已完成测试目的
- 不再需要保留在项目根目录
- 如需参考可查看归档文件

### JSON 示例
- 测试过程中生成的示例数据
- 已分析并文档化
- 保留作为参考

## 注意事项

- 归档文件不会影响项目运行
- 如需恢复，可从归档目录复制回原位置
- 建议定期清理过期的归档文件

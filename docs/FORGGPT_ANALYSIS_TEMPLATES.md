# ForgGPT 分析模板（Analysis Templates）

## 一、概述

本文档提供 ForgGPT 常用的分析模板，用户可以直接使用这些模板进行快速分析，也可以作为 AI 回答的参考格式。

## 三、使用方式

### 3.1 直接使用模板

用户可以直接复制模板中的问题，发送给 ForgGPT，AI 会基于当前数据进行分析。

### 3.2 自定义模板

用户可以根据自己的需求，修改模板中的参数（如时间范围、SKU列表等）。

### 3.3 组合使用

用户可以将多个模板组合使用，例如：
```
请先分析最近7天的GMV趋势，然后对比销量前10的商品，最后给出运营建议。
```

## 四、模板扩展

可以根据业务需求，添加更多分析模板：
- 促销活动效果分析
- 客户行为分析
- 区域销售分析
- 季节性趋势分析
- 竞品对比分析
